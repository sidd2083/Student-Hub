Build a full-stack web application called “Student Hub”.

IMPORTANT DESIGN RULE:
- UI must be clean, white, minimal, smooth like Instagram
- No futuristic AI glowing effects
- Use soft shadows, rounded cards, simple typography
- Focus on usability, not flashy design

TECH STACK:
- React / Next.js
- Tailwind CSS
- Firebase (Auth + Firestore)

---

# 🔐 AUTH SYSTEM

Use Google Sign-In (Firebase Authentication)

After login:
- ask user:
  - Full Name
  - Grade (9, 10, 11, 12)

Store user data:
{
  name,
  grade,
  role: "user" or "admin"
}
